# Task Manager Skill

This skill provides tools to manage development tasks using the `scripts/tasks.py` automation script.

## Tools
- `task_create`: Create a new task.
- `task_list`: List tasks.
- `task_update`: Update task status.
- `task_show`: Show task details.
- `task_context`: Show current context.
- `task_archive`: Archive a task.

See `docs/interop/tool_definitions.json` for full definitions.
